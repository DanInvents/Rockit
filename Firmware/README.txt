In this directory there are two folders:

real_coded_rotary_switch contains the firmware compatible with the version of Rockit featuring the real-coded rotary switch.

complementary_coded_rotary_switch contains the firmware compatible with the version of Rockit feauring the complementary-coded rotary switch.

To upload the firmware without any changes follow these steps:

1) Hold the pushbutton while connecting the flight computer to your PC with a microUSB cable

2) Drag and drop the .uf2 file to the directory that pops up when you connect Rockit to your PC.

3) Done
